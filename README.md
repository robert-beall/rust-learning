# Rust Basics Cheatsheet
## Basics 
### Compile a rust file
```
rustc <filename>.rs
```
### Run compiled rust code
#### Linux/MacOS
```
./<filename>
```
#### Windows
```
.\<filename>.exe
```
## Cargo
### Initialize a project with cargo
```
cargo new <project_name>
```
### Compile a cargo project
```
cargo build
```
### Run a cargo project 
```
cargo run
```
